# Installation
## Cloner le dépôt
```bash
# clonage du projet
git clone https://github.com/Tambour1/TicTacToe.git

# se placer à la racine du projet
cd TicTacToe
```
## Lancer le backend
```bash
# Se placer dans la répertoire à partir de la racine du projet
cd backend

# Installer les dépendances
npm install

# Lancer le backend
npm run start
```
## Lancer le frontend
```bash
# se placer à la racine du projet
cd TicTacToe

# Installer les dépendances
npm install

# Lancer le projet
npm run dev
```
L'application sera accessible à l'adresse local indiqué dans le terminal une fois la commande "npm run dev" lancée.

Exemples : 
- http://localhost:5173/
- http://localhost:5174/
